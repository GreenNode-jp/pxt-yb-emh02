# unpackJoystickValue

**joystickValue(Axis.XY)** の **`x+256y`** から **x** または **y** を取り出す。

```sig
ybemh02.unpackJoystickValue(0, ybemh02.JoystickAxis.X)
```

## Parameters

**packed**: **joystickValue**（**XY**）の戻り値

**component**: **X** / **Y**

## Returns

成分（整数想定）
